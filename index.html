Setting Up VS Code, Git & Connecting to GitHub ART 2440

1. Install Visual Studio Code

To install VS Code, follow these steps:

  1. Go to the VS Code website.
  2. Download the installer for your OS (Windows, macOS, Linux).
  3. Run the installer and follow the setup instructions.
  4. Once installed, open VS Code.

2. Install Git (Latest Version)

To install the latest version of Git:

  1. Visit Getting Started with Git --Git official website.
  2. Follow the instructions on the Git page for your OS
  3. Run the installer and keep the default options selected.
  4. After installation, open a terminal (Command Prompt or Git Bash) and type:
git --version

If Git is installed correctly, it will display the installed version.

3. Connect GitHub to VS Code

To connect VS Code with GitHub:

  1. Open VS Code and press Ctrl + Shift + P (or Cmd + Shift + P on macOS) to open the command palette.
  2. Type Git: Clone and select it.
  3. Enter the URL of your GitHub repository.
  4. Choose a folder on your computer to clone the repository.
  5. To authenticate, use the GitHub personal access token instead of a password.
  6. Once cloned, you can start coding and use Git from the VS Code terminal.
